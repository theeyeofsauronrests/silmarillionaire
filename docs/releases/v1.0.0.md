# Release Notes - v1.0.0

Release date: 2026-02-25

## Summary

Version `1.0.0` marks the first full MVP release of the platform. This release provides secure access control, hub-and-spoke roadmap visibility, project/team/people directory workflows, and admin governance for users, projects, and editing rights.

## Highlights

- Secure auth-gated application with role-aware access (`viewer`, `editor`, `admin`).
- Project detail pages with:
  - now/next/later roadmap board
  - key milestones
  - links and image references
  - team directory and flattened staffing list
- Editing workflows for authorized users, including drag-and-drop roadmap updates.
- Admin workflows for waitlist handling, role/status assignment, and project/editor management.
- Consolidated roadmap views for operational planning:
  - project-oriented grouping
  - core team-oriented grouping

## Hardening and Quality

- Strict TypeScript and linting checks.
- Unit tests and targeted E2E regression coverage.
- Security improvements for remote image URL validation and auth middleware behavior under misconfiguration.

## Upgrade Notes

- Version baseline updated to `1.0.0`.
- No data migration is required specifically for this version increment beyond the already tracked SQL migrations.
