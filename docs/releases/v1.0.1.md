# Release Notes - v1.0.1

Release date: 2026-02-25

## Summary

Version `1.0.1` is a focused UX and branding polish release after the initial 1.0 launch. It improves interaction clarity for save/edit workflows and strengthens visual consistency through updated token usage.

## Highlights

- Stronger brand-color prominence across navigation, headers, and directory cards.
- Updated font stack for headings/body/technical text to match the selected brand typography strategy.
- Better interaction confidence with save-state feedback on form submissions:
  - `Saving...` while requests are in-flight
  - `Saved` confirmation after completion

## Readability and Accessibility Improvements

- Removed transition glint overlay that introduced visual noise on page content.
- Increased contrast for CTA labels and form controls in dark mode.
- Improved active-navigation visibility and section hierarchy cues.

## Upgrade Notes

- Package version updated from `1.0.0` to `1.0.1`.
- No database migration changes are required for this release.
